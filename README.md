# ALARCON_M2Thesis_data_analysis
This repository gathers all R scripts and data files required for the data analysis conducted in my  M2 internship.

Direct selection on rec-1 was estimated in two different sets of competition experiments.

RMD files are all the scripts necessary for data analysis. The .R script "function_melting_analysis.R" gathers all functions used in the "analyse_melting_curve.Rmd" file.
This .R script is annotated to explain how all function work, and how the main function HRM() produces the data required for further rec-1 frequency estimations.
